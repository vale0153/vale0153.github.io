Image tag (img):
The image tag allows you to post an image on your website
Paragraph tag (p):
The paragraph tag allows you to post text on the website for people to read
Article tag (article):
The article tag represents a complete or self contained document or section in a website, such as a list
Unordered list tag (ul):
The unordered list tag allows you to write a list where the order is not a priority
Label tag (label):
The label tag represents a caption in a UI, such as if you were to select an option from a drop down menu
